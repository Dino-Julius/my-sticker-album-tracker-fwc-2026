# Guia de uso

La app esta pensada para usarla rapido desde celular o computadora.

## Dashboard

El Dashboard resume tu album con estas metricas:

- **Tengo**: estampas que ya tienes.
- **Faltantes**: estampas que todavia no tienes.
- **Repetidas**: estampas con copias extra.
- **Extras para cambiar**: copias disponibles para intercambiar.

## Registro

En **Registro** puedes buscar una estampa y cambiar su cantidad. Tambien esta **Registro rapido**, que permite pegar codigos, rangos o seleccionar varios codigos al mismo tiempo.

## Intercambio

En **Intercambio** puedes:

- copiar tu lista publica con **Me faltan** y **Mis repetidas**,
- comparar una lista de un amigo,
- registrar intercambios confirmados,
- crear apartados para intercambios futuros,
- editar, confirmar o cancelar apartados,
- revisar el historial confirmado.

## Cuenta

El panel de cuenta aparece compacto. Abre **Ver cuenta** para editar tu apodo, ver ultimas fechas, tocar **Sincronizar ahora** o cerrar sesion.

Si ves **Error al sincronizar**, toca **Ver detalle** para saber que fallo y **Reintentar** para probar de nuevo.
